# Face & Eye Detection Using OpenCV 👀

## 📌 Project Overview
This project detects **faces and eyes in real-time** using **OpenCV** and **Haar Cascade Classifiers**.  
It captures video from a webcam, detects faces and eyes, and draws bounding boxes around them.

This project can be used in applications like:

- 🔐 Security systems  
- 🧑‍🎓 Attendance systems  
- 🤖 Human-computer interaction  
- 📷 Real-time surveillance systems  

---

## 🛠 Tech Stack & Tools

**Programming Language**
- Python

**Libraries Used**
- OpenCV
- NumPy

**Algorithm**
- Haar Cascade Classifier (Pre-trained XML models)

---

## 📊 Project Approach

1. **Load Haar Cascade Models**
   - Load pre-trained XML files for face and eye detection.

2. **Capture Video Feed**
   - Access the webcam using OpenCV.

3. **Detect Faces and Eyes**
   - Use Haar Cascade classifiers to detect faces and eyes in each frame.

4. **Draw Bounding Boxes**
   - Draw rectangles around detected faces and eyes.

5. **Display Output**
   - Show the processed video stream in real time.
